# Simple Calculator

This is my first Python project. I built this after completing my Class 12 
with Computer Science. The idea was simple — build something that actually 
works and covers what I learned in school.

## What it does

- Addition, Subtraction, Multiplication and Division
- Power — raises one number to the power of another
- Handles divide by zero so it does not crash
- Keeps running until you choose to exit
- Works with decimal numbers too

## How to run it

Make sure Python 3 is installed, then run:

```
python calculator.py
```

Choose a number from 1 to 6, enter two numbers, and it gives the result.
Type 6 to exit.

## What I used

- Python 3.14.3
- pow() function for the power operation
- math module
